# B站直播工具 [BiliLiveTool]


该程序跨 .Net、 Python、 Html 、JavaScript 开发，考虑到环境问题，使用了独立打包发布，所以体积会显得略大。

其中悬浮窗口和网页可以切换 **“常规样式”** 和 **“聊天样式”**
